How to Build a Chatbot on Google Cloud

1. Introduction

Welcome to this workshop on building AI Agents with Google Cloud's Vertex AI Agent Builder. This guide will walk you through the process step-by-step, explaining each component along the way.

Prerequisites

  • A Google Cloud account
  • Basic understanding of Generative AI concepts
  • Familiarity with AI Agent fundamentals

What You'll Learn

  • Creating a cloud storage bucket for your data
  • Building an AI Agent using Vertex AI Agent Builder
  • Configuring your agent with custom instructions
  • Connecting your agent to data sources

Open Google Cloud Console.

2. Setting Up Google Cloud Storage

Step 1: Create a storage bucket

  • Navigate to Google Cloud Storage
  • Click on 'Create Bucket' button

Step 2: Configure your bucket

  • Enter a unique name for your bucket (lowercase letters only, no spaces)
  • Select Multi-region for Location type (ensure your agent uses the same region)
  • Leave other settings as default and click through to Create
  • Confirm the creation when prompted

Step 3: Upload your data

  • Click the Upload button to add your dataset folder. (

  • This data will be used to train your AI agent.

Step 4: Verify the upload

  • Ensure your data appears in the bucket
  • You have now successfully stored your training data on Google Cloud

3. Creating an AI Agent with Vertex AI

Now we'll create an AI agent that can use your uploaded data.

Step 1: Access Vertex AI

  • Go to Vertex AI Dashboard
  • If prompted, click CONTINUE AND ACTIVATE THE API

Step 2: Navigate to Vertex AI Search

  • Click on Vertex AI Search in the sidebar
  • Click the See All button on the right side
  • You'll be taken to the App Creation page
  • Click on the CREATE A NEW APP button
Note:

1. A new tab for Dialogflow Conversational Agents will open

2. If prompted to choose a Google Cloud Project, select the project 
   associated with your account

3. If using a new account, you may need to enable the Dialogflow API
  • If needed, enable the API manually by visiting the API page
  • In the Dialogflow page, click Create Agent

Step 3: Configure your agent

  • Enter the required information:
  • Company name (e.g., "YourCompany")
  • Agent name (e.g., "CustomerSupportAgent")
  • Location type: "Multi-region" (match your storage bucket)

Step 4: Connect your data store

  • After creating the agent, you'll be prompted to add data
  • Click Create Data Store
  • Select Cloud Storage as your data source
  • Click Browse and select the bucket you created earlier
  • Select your bucket and click Continue
  • Give your data store a name (e.g., "MyDataStore") and click Create
  • On the data page, click Create again to finalize

Step 5: App

  • Click on ‘Create’.
  • Congratulations! You've created your AI agent.
  • Click on your new app to open it.

Connect your data store tool

  • Scroll down to the Data store section
  • Configure the tool with these settings:
  • Tool name: ******
  • Type: Data store
  • Description: Use this tool if user's request contains a location that doesn't exist
  • Add your data store from the dropdown menu
  • Click Save to complete the configuration

Step 6: Configure your agent's behavior

  • In the left sidebar, select Playbook
  • Choose Routine to set up your agent's behavior
  • Configure your agent with the following settings:
Prompt:

You are an experienced Google recruiter chatbot, helping students and job seekers. 
When a user asks for job or HR info, answer with accurate, up-to-date details 
about roles, departments, or contacts at Google, based on pre-trained company
data. Always be clear, friendly, and guide users to application steps or 
further resources if needed.

and

Instructions:

- You are a helpful and experienced Google recruiter chatbot. 
- When users ask for job or HR information, answer with accurate details about 
roles, departments, or contacts at Google.
- Use your latest pre-trained data as well as ${TOOL: sampleTool} and 
${PLAYBOOK: samplePlaybook} when needed. 
- Always communicate clearly, supportively, and guide users to application 
steps or further official resources as appropriate. 
- Start with a friendly greeting, clarify requests, and provide detailed, 
actionable responses.

  • Click Add data stores to ensure your agent can access the information

4. Testing Your Agent

Once your agent is configured, you can test it by:

  • Clicking on the Test Agent tab in the Dialogflow interface
  • Typing sample questions related to your data
  • Verifying that responses contain accurate information from your data store
