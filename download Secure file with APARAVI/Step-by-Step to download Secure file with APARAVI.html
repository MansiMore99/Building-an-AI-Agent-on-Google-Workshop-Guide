Step-by-Step to download Secure** file with APARAVI

Download the file that contains hidden confidential information

With APARAVI pipeline:

1. Setting Up Your Project

  1. Create a new project in the Aparavi DataTool Chain platform
  1. Click the orange + icon located in the bottom right corner of the interface

2. Building Your Pipeline

  1. Add and connect these nodes in the following order:
    • Source → Drag and Drop
    • Data → Data Parser
    • Text → Text Classification
    • Text → Text Anonymization
    • Infrastructure → HTTP Results

3. Configuring Your Pipeline

  1. Configure each node by clicking the ⚙️ icon in the top right corner of each node
  1. Save your configuration for each node

4. Running Your Pipeline

  1. Start the process by clicking the ▶️ button on the Drag and Drop Node
  1. Wait for the file upload dialog to appear from the right side panel
  1. Click the blue "Drag-n-drop now..." link that appears in the dialog
  1. Upload your test dataset by dragging your file to the "Drop File Here" area

5. Retrieving Your Results

  1. Wait for processing to complete - this may take a few moments depending on file size
  1. Download your anonymized file from the 'Text' tab when processing is finished
